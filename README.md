
# Sitio académico — Cristian Meza

Este repositorio contiene los archivos del sitio para GitHub Pages.

## Estructura
```
/
├─ index.html
├─ investigacion.html
├─ docencia.html
├─ contacto.html
└─ assets/
   ├─ styles.css
   ├─ CV_CristianMeza.pdf  (reemplazar)
   └─ img/                 (coloca aquí tus imágenes)
```

## Publicar en GitHub Pages
1. Crea un repositorio público en GitHub (idealmente `usuario.github.io`).
2. Sube todo el contenido de esta carpeta.
3. Ve a **Settings → Pages**, selecciona la rama `main` y la carpeta `/ (root)`.
4. Tu sitio quedará activo en `https://usuario.github.io`.
